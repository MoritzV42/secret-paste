# Roadmap

Tracked work toward a stable, cross-platform v1.0 release.

- macOS Keychain backend (via `keyring`)
- Linux libsecret backend (via `keyring`)
- Bitwarden remote backend
- 1Password remote backend
- SSH/SQLite remote backend (for self-hosted teams)
- GitHub Actions CI (Windows / macOS / Linux matrix)
- pytest suite
- Animated GIF demo in README
- `pipx install` support
- Pre-commit hook for secret scanning (e.g. gitleaks / detect-secrets)
